Storage Dispatch Validation Test

This test validates that storage dispatch is working correctly by checking:

  • Storage charge/discharge prices are populated for all hours
  • Storage discharges when discharge price is high relative to charge price
  • Storage charges when charge price is low relative to discharge price
  • Storage dispatches regardless of arbitrage spread (no economic hurdle constraint)

Instructions to Run Test:

  1. Load the main HTML application (ETR_Dynamic_Cap_Expansion_Forecast_1.6.html)
  2. Upload required data files and configure storage capacity > 0
  3. Export results to CSV
  4. Upload the exported 8760 timeseries CSV file below
