Storage ELCC Top Days Stress Window Test

This test validates that the storage ELCC calculation uses a "Top Days" stress window approach: find the 10 days with the highest peak load, then for each day average Math.min(installedCap, storageSOC[idx-1]) over a 6-hour window (3 hours before the daily peak to 2 hours after). Final ELCC = (sumOfAvailablePower / 60) / installedCap.
