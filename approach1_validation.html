What is Approach 1?

Approach 1: SOC Before + Subtract Dispatch

  • Use State of Charge (SOC) at the beginning of the peak hour
  • Calculate available storage capacity based on this SOC
  • Subtract actual storage discharge to get unused capacity
  • Measures: "How much capacity was available but not used?"

Step 1: Get SOC Before Discharge

// Line 6851-6853 const socBeforePeak = (modelResults.storageSOC && peakLoadHour > 0) ? modelResults.storageSOC[peakLoadHour - 1] // SOC at END of previous hour = START of peak hour : 0; // If peak is hour 0, battery starts empty

Why peakLoadHour - 1? Because storageSOC[i] represents SOC at the end of hour i. So SOC at the start of hour N is storageSOC[N-1].

Step 2: Calculate Available Storage Capacity

// Line 6854-6855 const storageCap = installedCapacities['storage'] || 0; availableDispatchableAtPeak += Math.min(storageCap, socBeforePeak);

Reserve Margin = (Unused Capacity / Peak Load) × 100%
